# Replit Development Prompt: Intelligent Survey Data Cleaning Assistant

## Project Overview
Create a **Streamlit-based intelligent survey data cleaning assistant** for statistical agencies. This application must analyze each column individually and provide context-specific cleaning recommendations, integrated with AI assistance using Groq API's `llama-3.1-8b-instant` model.

## 🎯 Core Requirements (CRITICAL)

### 1. **Per-Column Intelligence - MOST IMPORTANT**
- **NEVER** apply the same cleaning method to all columns
- **MUST** analyze each column separately with different approaches
- **MUST** detect specific issues per column (missing values, outliers, inconsistencies)
- **MUST** recommend tailored solutions based on individual column characteristics
- **MUST** provide multiple cleaning options with clear pros/cons for each column

### 2. **Data Input & Configuration**
- Support CSV and Excel file uploads
- Automatic column type detection (beyond basic pandas dtypes)
- Interactive schema mapping interface
- JSON configuration import/export capability
- Real-time data preview with statistics

### 3. **Individual Column Analysis Requirements**
For EVERY single column, the system must detect:
- Missing value patterns and percentages
- Outliers using multiple methods: IQR, Z-score, Modified Z-score
- Data distribution characteristics (normal, skewed, bimodal, etc.)
- Data type inconsistencies and invalid entries
- Relationships with other columns

### 4. **Context-Specific Cleaning Strategies**
Based on individual column analysis, recommend different methods:
- **For Missing Values**: Mean/median imputation, KNN imputation, regression-based imputation, forward/backward fill, create missing category
- **For Outliers**: Winsorization, trimming, transformation, capping, removal
- **For Invalid Data**: Pattern-based correction, consistency checks, flagging for manual review
- **Selection Criteria**: Base recommendations on data type, missing percentage, outlier severity, and column relationships

### 5. **AI Integration Specifications**
- **Primary Model**: Groq API with `llama-3.1-8b-instant` (REQUIRED)
- **Fallback**: HuggingFace transformers for offline capability
- **Context Awareness**: AI must understand current dataset state and specific column being analyzed
- **Conversational Interface**: Natural language interaction for complex data cleaning questions
- **Educational Role**: Explain statistical concepts and methodology reasoning

## 🎨 User Interface Requirements

### Navigation Structure
Create a multi-page Streamlit application with:
- **Data Upload Page**: File upload with automatic schema detection
- **Column Analysis Page**: Detailed per-column analysis with visualizations
- **Cleaning Wizard Page**: Step-by-step cleaning process with user confirmations
- **Weight Management Page**: Statistical weighting application
- **Reports Page**: Auto-generated cleaning reports and documentation
- **AI Assistant Page**: Conversational AI helper integrated throughout

### Interactive Features Required
- Column selection dropdown for individual analysis
- Side-by-side before/after comparisons
- Multiple cleaning method buttons per column
- User confirmation dialogs before applying any changes
- Progress tracking showing cleaned vs. pending columns
- Undo/Redo functionality for all cleaning operations
- Real-time visualization updates

## 🤖 AI Assistant Requirements

### Behavioral Specifications
- Act as an expert survey data analyst
- Provide column-specific guidance (never generic advice)
- Explain statistical reasoning behind recommendations
- Compare different cleaning methods with pros/cons
- Consider survey methodology and sampling design implications
- Maintain conversation context about current dataset and analysis

### Required AI Capabilities
- **Column-Specific Questions**: "Why is KNN better than mean imputation for this specific column?"
- **Method Comparisons**: "Compare winsorization vs. trimming for these particular outliers"
- **Statistical Explanations**: "What does this missing data pattern suggest about data collection?"
- **Workflow Guidance**: "Which columns should I prioritize cleaning first and why?"
- **Impact Assessment**: "How will this cleaning method affect my survey estimates?"
